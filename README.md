# Quaggy API

## Description

API for Quaggy - Another Slack clone

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ docker-compose build
$ docker-compose up
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
